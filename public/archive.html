Setup notes
  1. Save each daily run as /archive/YYYY-MM-DD.json (ISO sort-friendly). Example: /archive/2025-10-13.json.
  2. (Optional but recommended) Generate /archive/manifest.json with an array like:
    [{"date":"2025-10-13","path":"/archive/2025-10-13.json","size": "42KB", "title":"Daily news scrape"}]
  3. This page will first try to load manifest.json. If missing, it will probe the last N days and list any files that exist.
